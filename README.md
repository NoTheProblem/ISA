# ISA
## Pokretanje
### Back
- Otvoriti Eclipse/Intellij
- Import project form ISA/pharmacy
- pom.xml -> Add as Maven Project
- PharmacyApplication.java -> Run
### Front
Pozicionirati se u ISA/pharmacy-front/
```sh
npm install -g @angular/cli
npm install
ng serve --port 4200
```
